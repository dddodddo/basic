# Flight slider - HTML+CSS

A Pen created on CodePen.io. Original URL: [https://codepen.io/alvaromontoro/pen/oNVEqgz](https://codepen.io/alvaromontoro/pen/oNVEqgz).

It uses:

- Trigonometrical functions
