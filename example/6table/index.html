table 태그 만들기(표 만들기)

html에서 표를 구현할 때

  • table 태그를 생성
  • tr 태그 생성(table row: 행, 줄)
  • td(table date) : 행의 내용, 열, 칸 table>tr>td
  • th(table heading):행, 열의 제목(굵은체, 가운데 정렬로 태어남) table>tr>th
