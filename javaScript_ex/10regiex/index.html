⛔ 정규표현식 (Regular expression의 약자 regiex(레직스)) 정규식 : 텍스트에서 우리가 원하는 특정한 패턴을 찾을 때 사용 (전화번호 형태의 패턴을 찾거나 웹사이트 주소형태의 패턴, 이메일 형태의 패턴을 찾을 때... 등등에 사용됨) : 정규식은 문자를 검사하고 싶을 때 사용된다. ⛔ 사용방법 : 정규식은 /로 시작하여 "나는 정규표현식"임을 나타낸다. : /우리가 찾고자 하는 패턴/ ==> /안에 우리가 찾고자 하는 패턴을 넣어주면 된다. : /regiex/i ==> i는 어떤 옵션에 따라 검색 할 건지 플래그를 활용함. ⛔ 문법 | : 또는 (or연산자) () : 그룹 [] : 괄호 안의 어떤 문자든 (?:) : 찾지만 기억하지는 않음(그룹에 대한 정보를 기억하지 않음.) ? : 없거나 있거나(zero or one) * : 없거나 있거나 많거나(zero or more) + : 하나 또는 많이(one or more) --> 없는 거 안찾음. {n} : n번 반복 {min,} : 최소 {min,max} : 최소, 최대 \b : 단어경계 \B : 단어경계가 아닌 것
